# NyWate_hub

🖤 NyBlackOut
NyBlackOut é um script avançado e multifuncional desenvolvido para o jogo Fling Things and People, oferecendo uma suíte completa de ferramentas para defesa, ataque, mobilidade e personalização visual. Com uma interface intuitiva e de alto desempenho, ele foi projetado para proporcionar o máximo de controle e vantagem ao jogador.

✨ Destaques Principais
🛡️ Defesa Imbatível
Anti-Grab Avançado: Múltiplos sistemas anti-agarramento (V1, V2, V3) com lógica inteligente que impede que outros jogadores te segurem.

Anti-Blobman: Neutraliza ataques com a criatura Blobman, incluindo detecção e destruição automática de detectores.

Anti-Explosão, Anti-Queimadura e Anti-Sticky: Proteção contra danos ambientais e ataques com cola.

Anti-Void e Anti-Lag: Evita quedas no vazio e reduz o lag causado por outros scripts.

Auto-Reset e Auto-Leave: Reinicia ou sai automaticamente para evitar banimentos em caso de detecção de "Flying".

🎯 Ataque e Controle de Alvos
Sistema de Alvos: Selecione facilmente qualquer jogador para aplicar as habilidades de ataque.

Kicks e Flings: Diversos métodos para lançar, arrastar e eliminar alvos (Loop Grab Kick, Spam Grab Kick, Fling Aura, etc.).

Ragdoll Spam: Ferramentas para desabilitar o "Ragdoll" do alvo, deixando-o vulnerável.

Pallet Hammer e Bomba de Teste: Técnicas avançadas de ataque em massa e silencioso.

Grab Personalizável: Ajuste a força e ative o "Kill Grab" para eliminar ao segurar.

🚀 Mobilidade e Exploração
Velocidade Personalizável: Aumente sua velocidade de movimento via CFrame.

Teletransporte: Teletransporte-se para o mouse ou para pontos específicos do mapa.

Platform TP: Crie uma plataforma no céu para se teleportar instantaneamente.

Blob Fly: Controle o Blobman como um veículo voador.

🎨 Personalização Visual
Auras e Efeitos: Mais de 10 efeitos visuais, incluindo Aura Orbital, Corpo de Raio, Aura de Fogo, Clone de Sombra, entre outros.

Chapéu Chinês e Trilha: Adicione um chapéu estiloso e uma trilha de cores.

ForceField e Skin Trail: Deixe seu personagem com uma aparência única.

Skyboxes e Efeitos Mundiais: Troque o céu, ative o modo "Full Bright", mude a hora do dia e muito mais.

Efeitos de Tela: Inclui efeitos de distorção e imagem de anime na tela.

⚙️ Ferramentas Adicionais
Detector de Pacotes: Monitore eventos de rede em tempo real para identificar ações de outros jogadores.

Keybinds Personalizáveis: Ative funções rapidamente com teclas de atalho.

Otimização de FPS: Botões para "Boost FPS" e "Delete Boost FPS" para melhorar o desempenho.

🖥️ Interface e Usabilidade
Design Inteligente: Interface limpa, com guias organizadas por categoria (Defesa, Alvo, Player, Misc, Visuals, etc.).

Configuração Automática: As configurações são salvas automaticamente para uso futuro.

Notificações em Tempo Real: Receba feedback instantâneo sobre ações e eventos importantes.

📦 Requisitos
Executor: Compatível com a maioria dos executores de script (Synapse X, Krnl, Script-Ware, etc.).

Jogo: Fling Things and People.

🚀 Como Usar
Copie o código do script.

Cole no seu executor.

Execute o script.

Use a tecla M (padrão) para abrir/fechar a interface gráfica.

⚠️ Aviso
Este script é fornecido apenas para fins educacionais e de entretenimento. O uso de scripts em jogos online pode violar os Termos de Serviço do jogo e resultar em banimento. Use por sua conta e risco.
